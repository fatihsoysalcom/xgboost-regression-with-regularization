# XGBoost Regression with Regularization

This example demonstrates how to use XGBoost for a regression task with synthetic data. It highlights key parameters related to gradient boosting and regularization, which are central to XGBoost's effectiveness in preventing overfitting and achieving high accuracy. The script generates simple data, trains an XGBoost model, and evaluates its performance.

## Language

`python`

## How to Run

1. Ensure you have Python installed.
2. Install required libraries: `pip install numpy scikit-learn xgboost`
3. Run the script: `python main.py`
